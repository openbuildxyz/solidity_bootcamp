# OwlSync项目介绍

## 1. 项目概述

- **项目名称**： Owlsync
- **项目简介**：人工智能驱动的Web3媒体聚合器
- **项目愿景**：实现实时信息同步和自动任务

## 2. 技术架构

- **Geth** - 1.13.x
- **通讯协议** - IPC/JRPC
- **校验码** - Hash256
- **节点存储** - 云服务器/本机IPFS
- **BNB** - BSC Testnet（Layer1）/BSC Mainnet（Layer2）

- **功能二**：智能问答
